About This Project

This project analyzes grid disruption data across the U.S. from 2014 to 2023, using Flask, interactive visualizations, and advanced machine learning models to uncover patterns, predict future outages, identify anomalies, and estimate outage durations.

Machine Learning Models Used

  • Prophet (Forecasting): Used in the Monthly Trend to predict future outage counts based on historical data, incorporating seasonality and trends.
  • Isolation Forest (Anomaly Detection): Applied in the Anomaly Detection section to identify unusual outage events, helping to pinpoint significant disruptions.
  • K-Means Clustering: Utilized in the Clustering section to group states with similar outage patterns, revealing regional trends.
  • RandomForest, XGBoost, LightGBM, and VotingClassifier (Outage Duration Prediction): Employed in the Predict Outage Duration section to estimate the duration of outages. These models are trained on a preprocessed dataset with SMOTE (Synthetic Minority Oversampling Technique) to handle class imbalance, ensuring robust predictions across different duration categories. The best model is selected based on the weighted F1-score.
